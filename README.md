## Hiking Sim

### Executable Downloads

#### Windows
- https://github.com/CSCD-475-game1/climb-sim/blob/dev/unity/climb-sim/Builds/Windows/abcdefg.zip 
- Unzip, run `climb-sim.exe`

#### Mac
- https://github.com/CSCD-475-game1/climb-sim/blob/dev/unity/climb-sim/Builds/Mac/abcdefg-mac.zip 
- Unzip
- Use `xattr -cr abcdefg.app` if Mac says it cannot verify the application

### Opening in Unity
- `git clone https://github.com/CSCD-475-game1/climb-sim.git`
- Open the project "climb-sim" under unity with Unity Editor 6.3 LTS
